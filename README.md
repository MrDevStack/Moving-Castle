# 🏰 Howl's Moving Castle - Interactive Landscape

An interactive, visually stunning 2.5D ambient web experience that brings the magical, wandering castle from Studio Ghibli's masterpiece to your browser. Built using **HTML5 Canvas**, **CSS3**, and **JavaScript**, this project features parallax scrolling, dynamic scenery, and responsive controls.

---

## 🎮 Features

* **Dynamic Castle Animation:** Watch the patchwork castle trek across the hills with custom physics for its mechanical legs and smoking chimneys.
* **Parallax Scrolling Background:** Layers of rolling green hills and distant alpine mountains that move at varying speeds to create deep visual immersion.
* **Interactive Environment:** Toggle mouse controls to influence the wind direction, scare the grazing sheep, or change the castle's walking pace.
* **Ambient Atmosphere:** A peaceful day/night cycle that shifts lighting seamlessly across the canvas.

## 🛠️ Controls

* `Toggle Mouse Controls` (Top Left Button): Switches between automatic cinematic camera panning and manual mouse-drag tracking.
* `Click/Tap`: Interact with elements in the foreground (like the shepherd or individual sheep).
